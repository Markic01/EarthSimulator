# Earth simulator
A very realistic Earth simulator implemented using OpenGL API

[demo video](https://youtu.be/qk8IruAV3hU)

---

## Controls

`W` - Move forward

`A` - Move left

`S` - Move backward

`D` - Move right

`Q` - Move downward

`E` - Move upward

`SCROLL` - Zoom
